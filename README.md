### AtCoder Beginner Contest回答環境
---

1. setupの**CONTEST_NUMBER**を回答したい大会番号に変更してターミナルで実行する

2. Solutionにコードを書いてTesterをターミナルで実行するとコードテストが可能

3. <font color ="Gray">残念ながら提出は手動</font>

何もかもはじめての場合は、Python入門 AtCoder Programming Guide for beginners(APG4bPython)がオススメです。
>https://atcoder.jp/contests/APG4bPython
